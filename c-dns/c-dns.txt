   Registration Procedure(s)

 Expert Review

   Expert(s)

 Unassigned

   Reference

   Note

 In version 1.0 of C-DNS [RFC-ietf-dnsop-dns-capture-format-10],
 there is a field describing attributes of the data recorded.  The
 field is a CBOR [RFC7049] unsigned integer holding bit flags.


   Available Formats
           [IMG]
           CSV

   Bit        Name                     Description                             Reference
   0    anonymised-data  The data has been anonymised.           [RFC-ietf-dnsop-dns-capture-format-10]
   1    sampled-data     The data is sampled data.               [RFC-ietf-dnsop-dns-capture-format-10]
   2    normalized-names Names in the data have been normalized. [RFC-ietf-dnsop-dns-capture-format-10]
   3-63 Unassigned

C-DNS Address Event Types

   Registration Procedure(s)

 Expert Review

   Expert(s)

 Unassigned

   Reference
           [RFC-ietf-dnsop-dns-capture-format-10]

   Note

 In version 1.0 of C-DNS [RFC-ietf-dnsop-dns-capture-format-10],
 there is a field identifying types of the events related to client
 addresses.  This field is a CBOR [RFC7049] unsigned integer.  There
 is a related optional field "ae-code", which, if present, holds an
 additional CBOR unsigned integer giving additional information
 specific to the event type.


   Available Formats
           [IMG]
           CSV

         Identifier                 Event Type               ae-code contents                    Reference
   0                      TCP reset                      None                      [RFC-ietf-dnsop-dns-capture-format-10]
   1                      ICMP time exceeded             ICMP code [icmpcodes]     [RFC-ietf-dnsop-dns-capture-format-10]
   2                      ICMP destination unreachable   ICMP code [icmpcodes]     [RFC-ietf-dnsop-dns-capture-format-10]
   3                      ICMPv6 time exceeded           ICMPv6 code [icmpv6codes] [RFC-ietf-dnsop-dns-capture-format-10]
   4                      ICMPv6 destination unreachable ICMPv6 code [icmpv6codes] [RFC-ietf-dnsop-dns-capture-format-10]
   5                      ICMPv6 packet too big          ICMPv6 code [icmpv6codes] [RFC-ietf-dnsop-dns-capture-format-10]
   6-18446744073709551615 Unassigned
