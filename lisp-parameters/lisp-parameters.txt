LISP ACT values

   Registration Procedure(s)

 IETF Review or IESG Approval

   Reference
           [RFC6830][RFC-draft-ietf-lisp-rfc6833bis]

   Available Formats
           [IMG]
           CSV

   Value           Action                                        Description                                              Reference
   0     No-Action                   The map-cache is kept alive and no packet encapsulation occurs.      [RFC6830][RFC-draft-ietf-lisp-rfc6833bis]
   1     Natively-Forward            The packet is not encapsulated or dropped but natively forwarded.    [RFC6830][RFC-draft-ietf-lisp-rfc6833bis]
   2     Send-Map-Request            The packet invokes sending a Map-Request.                            [RFC6830][RFC-draft-ietf-lisp-rfc6833bis]
   3     Drop/No-Reason              A packet that matches this map-cache entry is dropped. An ICMP       [RFC6830][RFC-draft-ietf-lisp-rfc6833bis]
                                     Unreachable message SHOULD be sent.
   4     Drop/Policy-Denied          A packet matching this Map-Cache entry is dropped because the target [RFC-draft-ietf-lisp-rfc6833bis]
                                     EWID is policy-denied by the xTR of the mapping system.
                                     Packet matching the Map-Cache entry is dropped because the
   5     Drop/Authentication-Failure Map-Request for the target EID fails an authentication check by the  [RFC-draft-ietf-lisp-rfc6833bis]
                                     xTR or the mapping system.
   6-7   Unassigned
