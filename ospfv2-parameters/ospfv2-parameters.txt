   Registries included below

     * OSPFv2 Options Registry (8 bits)
     * OSPF Packet Type (both v2 and v3)
     * OSPFv2 Link State (LS) Type
     * OSPFv2 Router LSA Link Type (Value 1)
     * OSPF Router Information (RI) TLVs
     * OSPF Router Informational Capability Bits
     * OSPFv2 Router Properties Registry
     * OSPFv2 Grace LSA Top Level TLV
     * Path Computation Element (PCE) Capability Flags
     * OSPFv2 Instance IDs
     * OSPF Extended Prefix Opaque LSA TLVs
     * OSPF Extended Prefix TLV Sub-TLVs
     * OSPF Extended Prefix TLV Flags
     * OSPF Extended Link Opaque LSA TLVs
     * OSPF Extended Link TLV Sub-TLVs
